本仓库提供 FDI 模组的示例代码，用于帮助用户快速上手模组，减少学习成本。
模组到手之后，建议先看其验证流程，熟悉上位机的简单使用后再看相关示例代码。
通过上位机，用户可以快速确认各项配置是否成功生效，并查看当前模组正在发送的所有数据包。
若用户需要获取数据包的详细结构与字段说明，请参阅模组的官方文档。

快速上手项目：
1.工程介绍
在所有的例程代码中，MCU中的FDILink_USART工程是相对简单并且全面的。
工程默认情况下是演示通过串口对模组进行配置，配置代码在main函数中，可以在FDI_config.h里面看到封装好的配置函数，仅通过串口通信可以配置模组参数。
数据帧的解码在串口的接受回调函数中实现，用户可以在FDI_receive.c里面，定义结构体全局变量在keil的watch查看或直接使用。
串口模式下还支持数据帧请求模式，在main.c里面相关代码被注释，用户可以可自行启用测试，执行后会返回连续10个数据帧，间隔0.1s。
若用户需要外部输入自己的数据来提高精度，代码中也有相关函数进行演示，用户可以可自行启用测试。

2.用户使用
用户准备一个stm32f103开发板，烧录FDILink_USART工程中的代码，PA9(USART1_TX)引脚接上模组的COM1的RX,PA10(USART1_RX)引脚接上模组的COM1的TX,
供电运行后，用户可以在上位机查看配置有无生效，在keil的watch窗口查看数据帧是否正常解码。
